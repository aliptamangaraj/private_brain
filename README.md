# 🧠 Private Brain

A personal knowledge management web application designed to organize thoughts, learning, and creativity in one place.

## Features
- Study tracking
- Knowledge notes
- Idea management
- Writing drafts
- Whiteboard for visual thinking
- Backup & restore system

## Tech Stack
- HTML
- CSS (Custom design system)
- JavaScript (DOM + LocalStorage)
